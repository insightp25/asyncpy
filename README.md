# 파이썬 동시성 프로그래밍을 활용한 도서 빅데이터 아카이빙 애플리케이션

### 주요 기능
1. 네이버, 카카오 오픈 API를 활용한 도서 데이터 검색
2. 도서 데이터 웹 크롤링, 스크래핑
3. 도서 데이터 시각화
3. 도서 데이터 저장

### 활용 기술 및 지식
- 동시성/병렬성 프로그래밍 코드
- 파이썬 코루틴, 비동기 프로그래밍
- 파이썬 멀티 스레딩, 멀티 프로세싱 기법
- 프로세스와 스레드 관련 OS 지식
- MongoDB 활용 비정형 빅데이터 적재
- FastAPI를 활용한 서버 개발, 비동기 게이트웨이 Uvicorn 활용
- AWS 클라우드 서버 구축
- 웹 데이터 크롤링, 시각화
